# print と四則演算

競プロでは「答えを出力する」のが最終ゴール。  
まず `print` の使い方と、基本的な計算をおさえよう。

---

## print の基本

```python
print("Hello, World!")  # Hello, World!
print(42)               # 42
print(3.14)             # 3.14
```

`print()` は何でも出力できる。文字列は `"..."` または `'...'` で囲む。

---

## 複数の値を出力する

```python
print(1, 2, 3)        # 1 2 3（空白区切りで出力される）
print("ans:", 42)     # ans: 42
```

デフォルトは空白区切り。`sep` で変更できる：

```python
print(1, 2, 3, sep=",")   # 1,2,3
print(1, 2, 3, sep="")    # 123
```

---

## 改行なしで出力する

```python
print("Hello", end="")
print("World")
# HelloWorld
```

`end` のデフォルトは `"\n"`（改行）。`end=""` にすると改行しない。

---

## 四則演算

| 演算子 | 意味     | 例           | 結果 |
|--------|----------|--------------|------|
| `+`    | 足し算   | `3 + 2`      | `5`  |
| `-`    | 引き算   | `3 - 2`      | `1`  |
| `*`    | 掛け算   | `3 * 2`      | `6`  |
| `/`    | 割り算   | `7 / 2`      | `3.5`|
| `//`   | 整数除算 | `7 // 2`     | `3`  |
| `%`    | 余り     | `7 % 2`      | `1`  |
| `**`   | べき乗   | `2 ** 10`    | `1024`|

```python
print(3 + 2)    # 5
print(3 - 2)    # 1
print(3 * 2)    # 6
print(7 / 2)    # 3.5  ← 小数になる
print(7 // 2)   # 3    ← 小数点以下切り捨て
print(7 % 2)    # 1    ← 余り
print(2 ** 10)  # 1024
```

---

## 整数除算と余りについて

競プロでよく使う。「偶数・奇数の判定」や「グループ分け」に活躍する。

```python
print(10 % 2)   # 0 → 偶数
print(11 % 2)   # 1 → 奇数
print(13 % 5)   # 3 → 5で割った余り
```

---

## 計算の優先順位

数学と同じ。`*` `/` が先、`+` `-` が後。  
カッコ `()` で優先順位を変えられる。

```python
print(2 + 3 * 4)    # 14（3*4 が先）
print((2 + 3) * 4)  # 20（カッコが先）
```

## 例題：ABC086A

**問題**  
2つの整数 A, B が与えられる。A×B が偶数なら `Even`、奇数なら `Odd` を出力せよ。

**入力例**
```
3 4
```

**出力例**
```
Even
```

**コード**
```python
a, b = map(int, input().split())
print("Even" if a * b % 2 == 0 else "Odd")
```

## よく使うパターン一覧

| 数値を出力               | `print(42)` |

| 区切り文字を変える       | `print(a, b, sep=",")` |
| 改行なしで出力           | `print(a, end="")` |
| 足し算                   | `a + b` |
| 引き算                   | `a - b` |
| 掛け算                   | `a * b` |
| 割り算（小数）           | `a / b` |
| 割り算（整数）           | `a // b` |
| 余り                     | `a % b` |
| べき乗                   | `a ** b` |
